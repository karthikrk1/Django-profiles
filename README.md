# Profiles REST API

REST API for managing user Profiles

## Requirements
Vagrant >= 1.9.8
VirtualBox 
Git
Atom (or any other editor)
Mod Headers (Google Chrome plugin)

[Vagrant](https://www.vagrantup.com/downloads.html)
